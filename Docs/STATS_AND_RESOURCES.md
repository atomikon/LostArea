# Stats & Resources (FINAL+)

## Core Stats
- Health (absolute value)
- Chakra (100%)
- Stamina (100%)
- Physical Defense
- Magical Defense

## Damage Rules
- Attacks → Physical damage only
- Skills → Magical damage only
- Defenses reduce incoming damage of their type

## Chakra
- Chakra is represented as 100%
- Skills consume chakra in percentages
- Base regeneration (recommended starting point):
  - 5% per second (out of combat)
  - optional: 2–3% per second in combat
- Equipment/buffs can:
  - increase chakra regeneration
  - reduce chakra costs

## Stamina
- Stamina is represented as 100%
- Used for:
  - Dodge
  - Wall Run
  - Grapple (Q) 15% per use
  - Block
- Base regeneration (recommended starting point):
  - 10% per second (idle/walk)
  - 5–7% per second while moving
  - optional: 5% per second in combat
- Equipment/buffs can:
  - increase stamina regeneration
  - reduce stamina costs

## Health
- Base regeneration (recommended starting point):
  - 1–2% of max HP per second out of combat only
  - no regen in combat (recommended)
- Equipment/buffs/items can:
  - increase max HP
  - increase HP regeneration
  - grant temporary shields or heals
